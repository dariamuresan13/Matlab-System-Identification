# System Identification Algorithms (MATLAB)

Collection of MATLAB implementations for system identification and dynamic system modeling. 
The project includes parameter estimation algorithms, signal generation methods, and model validation using both experimental datasets and DC motor simulation data.

• ARX model identification using least squares estimation
• Instrumental Variable (IV) method for parameter estimation
• FIR model identification using correlation analysis
• Nonlinear parameter estimation using the Gauss–Newton algorithm
• Generation of excitation signals (step, impulse, PRBS)
• Dynamic system modeling and validation using DC motor simulation data
• Model validation using prediction, simulation, RMSE, and fit metrics

## Technologies

• MATLAB  
• Least Squares Estimation  
• Instrumental Variable Methods  
• Gauss–Newton Optimization  
• Correlation-based FIR Modeling  
• PRBS Signal Generation
