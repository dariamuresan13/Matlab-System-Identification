# System Identification Algorithms (MATLAB)

Collection of MATLAB implementations for system identification and dynamic system modeling.  
The project explores multiple identification techniques, excitation signal generation, and model validation using both experimental datasets and DC motor simulation data.

---

## Algorithms Implemented

- **ARX Model Identification** using least squares estimation
- **Instrumental Variable (IV) Method** for improved parameter estimation
- **FIR Model Identification** using correlation analysis
- **Nonlinear Parameter Estimation** using the Gauss–Newton optimization algorithm

---

## Experiments and Signal Generation

- Step input experiments for dynamic system analysis
- Impulse-based system response estimation
- PRBS (Pseudo-Random Binary Sequence) excitation signals
- Data acquisition using a DC motor simulation model

---

## Model Validation

- Simulation and prediction comparison
- RMSE (Root Mean Square Error) evaluation
- Fit percentage calculation
- Model comparison using MATLAB System Identification Toolbox

---

## Technologies

- MATLAB
- Least Squares Parameter Estimation
- Instrumental Variable Methods
- Gauss–Newton Optimization
- PRBS Signal Generation
- Dynamic System Modeling
